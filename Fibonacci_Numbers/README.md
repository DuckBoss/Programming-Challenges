# Usage - 
- Provide 2 starting integers and a max count integer in the CLA.

# Example - 
(Fibonacci sequence with '1' and '2' as starting numbers, up to 10 terms)
- CLA: 1 2 10

# Output -
- Given Starting Numbers: '1' and '2'
- Given Max Term Count: '10'
